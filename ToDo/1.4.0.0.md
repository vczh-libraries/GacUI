### INativeController::AutomationService

Add a `AutomationService` to `INativeController`.
Each `Setup*` function is responsible to substitute it when possible.
`Stop` can be called to shut down the service manually.

`StartWindowsHttpAutomationService` function to start a `HttpServerApi` on `AutomationService`.
  - `CppTest*` and `RemotingTest*` start it automatically when in `VCZH_MSVC && DEBUG`, listening to `localhost:8888/<ProjectName>/`, concatinating with following URLs:
  - `Control` or `ControlVerbose`, returning JSON.
  - `Dom`, returning JSON.
  - `IO` with commands in body in JSON `string[]`, returning empty body immediately, not waiting for the execution.
    - When the body begins with `!Type:`, all the rest of raw text are typed directly.
    - Implemented in `Windows`.

UI Automation won't work in locked screen, this feature is crucial for coding agent.
- Core exposes control tree, need to mark `MultiWindow|Hosted|Headless`:
  - `MultiWindow` when not hosted and not remote protocol. Multiple `<Window/>` or `<Popup/>` tags appear under `<GacUI/>`.
  - `Hosted` when hosted and not remote protocol. Main `<Window/>` under `<GacUI/>` and everything under the main window.
  - `Headless` when remote protocol.
  - Only lists visible windows. When in `MultiWindow` each window has an ID which is the pointer to `INativeWindow`.
- Renderer exposes DOM tree.
- Non `Headless` core or renderer listen to operations that sends messages to the underlying native controller.
  - Operations is on the main window.
  - Operations is on DPI-adjusted logical coordinate on the client area.
  - When in hosted mode or to renderer, the main window ID could be omitted. Since ID will be the pointer, it should call `INativeController::WindowService` to find out if it exists.
- General implementation should rely on `DumpWindowClientArea` and `RunIOCommandOnNativeWindow`.

- Update `Guidelines` to mention it in GUI applications, encouraging attaching debugger, turning on HTTP server when it does not.
  - Prepared powershell scripts to enumerate windows and reveal if there is any system dialog box.
- Enable it for `macOS`.
