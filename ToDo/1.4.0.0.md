## Milestone

### Release Issues

- `Codepack` fail to execute on Linux because of path delimiter issues.
  - Use FilePath to normalize file name before reading.

### VlppOS

- A channel framework focusing on local multi-process information interchange.
  - UTF <-> UTF
  - Copy from GacUI
    - No `ProcessEvents` and `ExecutablePath`.
  - `SendToClient` and `BroadcastFromClient` to all clients that available to this channel.
  - `Connected` and `Disconnected` and `Stopped` events.
- A server/client framework managing multiple channels.
  - Client declares available channels at connection.
  - Default server/client implementation offering flexible data type features.
    - Send/Receive function takes separated chanel and data argument.
- Predefined implementations
  - Windows
    - stdio 
    - Named Pipe (any)
    - HTTP (UTF-8)
      - Connection: `requestUrl;responseUrl`
      - Body: `(clientId)?;channelName;str`
  - Linux/macOS
    - stdio
    - Socket (any)
